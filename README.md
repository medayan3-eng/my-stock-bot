# Murphy Screener — הוראות הפעלה

זהו **סורק/מסמן**, לא בוט מסחר אוטומטי. הוא סורק רשימת מניות, נותן ציון 0-100 לכל אחת, ומסביר בטקסט למה קיבלה את הציון. אתה מחליט מה לעשות עם התוצאות.

## התקנה (חד פעמי)
```
pip install yfinance pandas numpy
```

## הרצה
```
python murphy_screener.py                     # סורק את כל ה-S&P 500 (לוקח כמה דקות)
python murphy_screener.py AAPL MSFT NVDA       # סורק רק את הטיקרים האלו
python murphy_screener.py --file mylist.txt    # קובץ עם טיקר בכל שורה
python murphy_screener.py --top 20             # מציג רק את 20 המניות המדורגות הכי גבוה
```

הפלט: טבלה במסך + קובץ `screener_results.csv` עם כל התוצאות כולל ההסברים המלאים.

## מה נכלל בציון (0-100)
| רכיב | ניקוד | מבוסס על |
|---|---|---|
| מגמת עלייה 52 שבועות + מעל ממוצע 50/200 | עד 25 | Technical Analysis of the Financial Markets |
| צמידות לממוצע 50 עם נפח גבוה (הזדמנות) | עד 10 | שם |
| מצב רצועות בולינגר (סקוויז / נגיעה ברצועה תחתונה) | עד 10 | שם |
| נר יפני חיובי (פטיש, בליעה, כוכב בוקר וכו') | 10 | Japanese Candlestick Charting Techniques |
| נפח מסחר חריג (כניסת כסף מוסדי) | עד 10 | Technical Analysis of the Financial Markets |
| RSI | עד 10 | שם |
| MACD | עד 10 | שם |
| מובילות סקטוריאלית (1ח/1ח/3ח/12ח יחסית ל-SPY) | עד 10 | Intermarket Analysis |
| התאמה למשטר האינטרמרקט (אינפלציה/דיפלציה) | 5 | Intermarket Analysis |

כל מניה מקבלת גם **סטופ לוס** (מתחת לתמיכה האחרונה / ממוצע 50, הנמוך מביניהם) ו**מחיר יעד** (יחס סיכוי:סיכון ~2.5, או השיא של 52 השבועות אם גבוה יותר).

## הערות חשובות
- הסקריפט מושך נתונים חיים דרך `yfinance` — צריך אינטרנט בזמן ההרצה.
- רשימת ה-S&P 500 נמשכת מוויקיפדיה; אם זה נכשל, יש רשימת גיבוי קטנה — עדיף להעביר קובץ טיקרים משלך (`--file`).
- זה כלי איתור הזדמנויות, לא ייעוץ השקעות. כל החלטת מסחר היא שלך.
- אפשר לשנות בקלות את המשקלים/הספים בראש הקובץ (למשל `VOLUME_SPIKE_MULT`, `NEAR_MA50_PCT`).
